# Musicfy-Spotify-Clone

Musicfy is a spotify clone where we can listen songs.

## Frontend
Start with:
```
install Node.js
```
Followed by:
```
installing live-server extension
```
Note: Please make sure, you have **node.js** and **live-server** presiding in your system.
